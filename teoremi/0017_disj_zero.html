Disj Zero

\(\vdash \forall p.\ p \vee \top \Leftrightarrow \top\)

disj_zero_thm
// |- !p. p \/ true <=> true

let p = "p:bool" |> parse_term
let th1 = disj2_rule p truth_thm                        // |- p \/ true
let th2 = eqt_intro_rule th1                            // |- p \/ true <=> true
let th3 = gen_rule p th2                                // |- !p. p \/ true <=> true

val disj2_rule: tm: term -> th: thm -> thm
<summary> This is the disjunction introduction rule for the RHS. It disjoins the supplied boolean term to LHS of the supplied theorem. `p` A |- q ------------ A |- p \/ q Si veda anche: disj1_rule, disj_cases_rule, mk_disj2_rule. </summary>
val truth_thm: thm
<summary> |- true </summary>
val th2: thm
val eqt_intro_rule: th: thm -> thm
<summary> This is the truth equivalence introduction rule. It takes any theorem, and returns the theorem stating that the conclusion is equivalent to truth, under the same assumptions. A |- p --------------- A |- p &lt;=&gt; true See also: eqt_elim_rule, eqf_intro_rule. </summary>
val th3: thm
val gen_rule: v: term -> th: thm -> thm
<summary> gen_rule : term -&gt; thm -&gt; thm This is the universal introduction rule. It universally quantifies the supplied theorem with the supplied binding variable, under the same assumptions. The binding variable must not occur free in the assumptions. `x` A |- p [ "x" not free in `A` ] ------------ A |- !x. p See also: list_gen_rule, spec_rule, mk_forall_rule. </summary>
